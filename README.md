Netflix Data Trend Analysis 📊
Project Overview

This project focuses on analyzing Netflix dataset trends using Microsoft Excel. The analysis helps understand content distribution, genre popularity, release trends, ratings, and country-wise content availability through interactive dashboards and visualizations.

Key Insights
Movies dominate Netflix content compared to TV Shows.
Content production increased rapidly after 2015.
Drama and Comedy are among the most popular genres.
The United States contributes the highest number of titles.
TV-MA is one of the most common content ratings.
